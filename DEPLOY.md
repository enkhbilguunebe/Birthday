# Correct GitHub Pages deployment

Upload the **contents of this folder directly to the repository root**.

The root of the repository must look like this:

- `index.html`
- `.nojekyll`
- `css/`
- `js/`
- `assets/`

Do not upload a single parent folder named `birthday-universe` around these files.

Then open:

1. Repository **Settings**
2. **Pages**
3. Source: **Deploy from a branch**
4. Branch: `main`
5. Folder: `/ (root)`
6. Save

Wait about 1–3 minutes, then hard-refresh the published page.
